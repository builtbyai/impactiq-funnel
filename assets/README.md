# ImpactIQ Assets

This directory contains all images, logos, and media files for the ImpactIQ roofing funnel website.

## Required Images

### Logo Files
- **impactiq-logo.png** - Main logo (recommended: 200x60px, transparent background)
- **impactiq-logo-animated.gif** - Animated logo for hero section intro (recommended: 400x400px)

### Image Specifications

All images should be optimized for web use:
- Format: PNG for logos (transparency), JPG for photos, WebP for modern browsers
- Optimization: Use tools like TinyPNG, ImageOptim, or Squoosh
- Naming: Use lowercase with hyphens (e.g., `roof-damage-photo.jpg`)

## Current Image Fallbacks

The website is currently configured with graceful fallbacks:
- Missing logos will hide automatically without breaking layout
- Font Awesome icons are used as placeholders for feature icons
- Gradient backgrounds are used instead of photo backgrounds

## How to Add Images

1. Place your optimized images in this directory
2. Update file references in `index.html` if using different names
3. For React version, images will be imported in component files

## Image Optimization Tools

- [TinyPNG](https://tinypng.com/) - PNG/JPG compression
- [Squoosh](https://squoosh.app/) - Google's image optimizer
